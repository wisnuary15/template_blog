## Neuron Artificial dan Jaringan Saraf Tiruan (Artificial Neural Networks/ANN)

Pada dasarnya, Deep Learning terinspirasi dari struktur dan fungsi otak manusia. Neuron dalam otak manusia memiliki kemampuan untuk menerima sinyal, memproses informasi, dan mengirimkan sinyal keluar. Dalam Deep Learning, kita menggunakan Neuron Artificial, yang digunakan untuk memodelkan jaringan saraf tiruan.

Contoh kode untuk membuat Neuron Artificial sederhana:
```js
class Neuron:
    def __init__(self, input_size):
        self.weights = [0.0] * input_size
        self.bias = 0.0

    def activate(self, inputs):
        total = sum(w * x for w, x in zip(self.weights, inputs)) + self.bias
        return 1.0 if total >= 0.0 else 0.0
```
## Perceptron

Perceptron adalah bentuk sederhana dari jaringan saraf tiruan yang dapat melakukan klasifikasi biner. Ini terdiri dari satu lapisan neuron input dan lapisan neuron output.

Contoh kode untuk perceptron:
```js
class Perceptron:
    def __init__(self, input_size):
        self.neuron = Neuron(input_size)

    def predict(self, inputs):
        return self.neuron.activate(inputs)
```

## Multilayer Perceptron (MLP)

MLP adalah jaringan saraf tiruan dengan lebih dari satu lapisan tersembunyi. Ini memungkinkan model untuk mempelajari fitur yang lebih kompleks.

Contoh kode untuk MLP:
```js
class MLP:
    def __init__(self, input_size, hidden_sizes, output_size):
        self.hidden_layers = []
        for size in hidden_sizes:
            self.hidden_layers.append([Neuron(size) for _ in range(size)])
        self.output_layer = [Neuron(output_size) for _ in range(output_size)]

    def predict(self, inputs):
        hidden_outputs = [neuron.activate(inputs) for neuron in self.hidden_layers[0]]
        output_inputs = [output.activate(hidden_outputs) for output in self.output_layer]
        return output_inputs
```
