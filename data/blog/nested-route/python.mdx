Python adalah bahasa pemrograman yang populer dan sering digunakan baik untuk pemula maupun pengembang berpengalaman. Dengan sintaksis yang intuitif dan berbagai pustaka yang kuat, belajar Python bisa menjadi pengalaman yang menyenangkan. Berikut adalah panduan tentang cara cepat belajar Python dengan beberapa contoh kode untuk memudahkan pemahaman Anda.

## 1. Persiapan Awal
Sebelum kita masuk ke contoh kode, pastikan Anda telah menginstal Python di komputer Anda. Anda dapat mengunduhnya dari [situs resmi Python](https://www.python.org/downloads/) dan mengikuti panduan instalasi yang diberikan.

## 2. Hello, World!
Mari kita mulai dengan contoh kode paling sederhana, yaitu mencetak "Hello, World!" di layar. Ini adalah langkah pertama yang umum dilakukan saat mempelajari bahasa pemrograman baru.

\`\`\`python
print("Hello, World!")
\`\`\`

## 3. Variabel dan Tipe Data
Python memiliki beberapa tipe data dasar seperti integer, float, string, dan boolean. Berikut adalah contoh penggunaan variabel dan tipe data:

\`\`\`python
// Variabel dan Tipe Data
nama = "John"
umur = 25
tinggi = 175.5
is_pelajar = True

print("Nama:", nama)
print("Umur:", umur)
print("Tinggi:", tinggi)
print("Pelajar:", is_pelajar)
\`\`\`

## 4. Struktur Kendali
Python menggunakan pernyataan if, else, dan elif untuk pengambilan keputusan. Berikut contoh penggunaannya:

\`\`\`python
// Struktur Kendali
nilai = 85

if nilai >= 90:
    print("Nilai: A")
elif nilai >= 80:
    print("Nilai: B")
else:
    print("Nilai: C")
\`\`\`

## 5. Looping
Looping digunakan untuk mengulang tindakan tertentu. Contoh di bawah ini menggunakan loop for untuk mencetak angka 1 hingga 5.

\`\`\`python
// Looping
for i in range(1, 6):
    print(i)
\`\`\`

## 6. Fungsi
Fungsi digunakan untuk mengelompokkan kode menjadi blok yang dapat dipanggil kapan saja. Berikut contoh sederhana:

\`\`\`python
// Fungsi
def sapa(nama):
    print("Halo,", nama)

sapa("Alice")
\`\`\`

## 7. List dan Looping
List adalah kumpulan data yang dapat diakses melalui indeks. Looping dapat digunakan untuk mengakses dan memanipulasi list.

\`\`\`python
// List dan Looping
buah = ["apel", "jeruk", "pisang", "anggur"]

for buah in buah:
    print(buah)
\`\`\`

## 8. Membaca Input Pengguna
Anda dapat membaca input dari pengguna menggunakan fungsi \`input()\`.

\`\`\`python
// Membaca Input Pengguna
nama = input("Masukkan nama Anda: ")
print("Halo,", nama)
\`\`\`

## 9. Contoh Proyek: Kalkulator Sederhana
Berikut adalah contoh sederhana kalkulator dengan fungsi dasar.

\`\`\`python
// Kalkulator Sederhana
def tambah(a, b):
    return a + b

def kurang(a, b):
    return a - b

def kali(a, b):
    return a * b

def bagi(a, b):
    return a / b

angka1 = float(input("Masukkan angka pertama: "))
angka2 = float(input("Masukkan angka kedua: "))

print("Hasil Penjumlahan:", tambah(angka1, angka2))
print("Hasil Pengurangan:", kurang(angka1, angka2))
print("Hasil Perkalian:", kali(angka1, angka2))
print("Hasil Pembagian:", bagi(angka1, angka2))
\`\`\`

## 10. Menggunakan Pustaka Eksternal
Salah satu keunggulan Python adalah ekosistem pustaka yang kaya. Contoh di bawah ini menggunakan pustaka \`math\` untuk menghitung akar kuadrat.

\`\`\`python
// Menggunakan Pustaka Eksternal
import math

angka = 25
akar_kuadrat = math.sqrt(angka)
print("Akar Kuadrat dari", angka, "adalah", akar_kuadrat)
\`\`\`
